## Thymeleaf 的一些主要标签和函数
- `th:text` 显示文本
- `th:utext` 和 `th:text` 的区别是针对 "unescaped text"
- `th:attr` 设置标签属性
- `th:if` 和 `th:unless` 条件判断语句
- `th:each` 循环语句
- `th:style` 设置样式
- `th:onclick` 点击事件
- `th:href` 链接地址
- `th:inline` 定义 js 脚本可以使用变量
- `th:action` 表单提交地址
- `#dates` 日期函数
- `#calendars` 日历函数
- `#numbers` 数字函数
- `#strings` 字符串函数
- `#objects` 对象函数
- `#bools` 逻辑函数
- `#arrays` 数组函数
- `#lists` 列表函数
